## What This Repository Is

Three superconductor specifications derived through R³ Composite State Dissolution. ## The Profit Points

```
SC-02 PdHx — Frozen ~1990s
Problem: Bulk loading irreproducible. Field abandoned.
Fix: Epitaxial film + electrochemical loading.
Eliminates grain boundaries, phase segregation,
and uncontrolled stoichiometry simultaneously.

SC-03 Iron Pnictides — Frozen ~2012
Problem: Chemical doping ceiling accepted as physical limit.
Fix: Optoelectronic doping via 405 nm illumination.
Adds carriers without adding disorder.
Reversible. Tunable. No ceiling.

SC-04 Carbon Nanotubes — Frozen ~2000s
Problem: Contacts + chirality + defects
each partially solved. Never all together.
Fix: (10,10) armchair + C-12 isotopic purity
+ Pb/NbN superconducting contacts.
All three problems eliminated simultaneously.
```

## How to Verify

Any materials science laboratory with thin film deposition, XRD, four-point probe with cryostat, and SQUID magnetometer can verify these specifications.

Follow the synthesis protocol in each material folder. Report all results — positive and negative — to `verification/submissions/`. Every result will be published here credited to your institution.
