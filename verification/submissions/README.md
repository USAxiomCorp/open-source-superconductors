# Verification Submissions

Submit results from any attempt to verify SC-02, SC-03, or SC-04.

## How to Submit

Name your folder:
```
[SC-NUMBER]-[MATERIAL]-[YYYYMMDD]-[INSTITUTION]-[RESULT]
```

Results: CONFIRM / DENY / INCONCLUSIVE

## Required Contents

```
report.md          — structured summary
raw_data/          — all raw data unprocessed
analysis/          — scripts and outputs  
photos/            — setup and sample photos
```

## Report Template

```markdown
# Verification Report
**Material:** SC-0X
**Institution:** 
**Operator:** 
**Date:** 
**Result:** CONFIRM / DENY / INCONCLUSIVE

## Synthesis
[What was done. Any deviations from protocol.]

```

## Policy

All results published here without modification.
Positive and negative results equally valued.
